## What is this?

Paste any blog post, article, or long-form content. Get back:

- 🐦 **Twitter/X threads** with hooks and engagement
- 💼 **LinkedIn posts** in storytelling format  
- 📸 **Instagram captions** with hashtags
- 📧 **Email newsletters** with subject lines
- 🎵 **TikTok scripts** with timestamps
- 📝 **Blog summaries** optimized for SEO
- 🎥 **YouTube descriptions** with chapters
- 🎙️ **Podcast show notes**

Each output is **platform-native** — not just rewritten text, but content structured for how each platform actually works.

## Why?

Creating content is hard. Distributing it across 8 platforms is **tedious**. Most creators write one great post, share it on one platform, and let the rest rot.

Repurpose AI eliminates the distribution bottleneck. Write once, publish everywhere.

## Quick Start

### Use the web app (easiest)
1. Go to [repurpose-ai.trycloudflare.com](https://journalists-conversation-sell-spider.trycloudflare.com)
2. Paste your content
3. Select formats
4. Click "Repurpose"
5. Copy results to each platform

### Use the API
```bash
curl -X POST https://journalists-conversation-sell-spider.trycloudflare.com/api/repurpose \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Your blog post text here...",
    "formats": ["twitter_thread", "linkedin_post", "instagram_caption"]
  }'
```

### Embed on your website
```html
<script src="https://journalists-conversation-sell-spider.trycloudflare.com/embed/widget.js" defer></script>
```

Adds a "Repurpose This" button to any page. [See demo →](https://journalists-conversation-sell-spider.trycloudflare.com/embed/demo.html)
